# Portfolio Website

Dette repository indeholder min personlige portfolio-website.

Siden er udviklet som en samlet præsentation af min faglige profil, mine kompetencer og udvalgte projekter. Formålet er at skabe et overskueligt og professionelt sted, hvor jeg kan formidle, hvem jeg er, hvad jeg arbejder med, og hvilke områder inden for software og IT jeg interesserer mig for.

Portfolioen består blandt andet af en forside, en side om mig, en CV-side og en projektside.

Projektet har samtidig været brugt som en praktisk ramme for at arbejde med udviklingsworkflow i praksis, herunder Git og GitHub-flow, branching, merges, responsivt design og deployment i Azure.

Løsningen er udviklet i Blazor med C#, HTML, CSS og JavaScript og bliver løbende videreudviklet.
